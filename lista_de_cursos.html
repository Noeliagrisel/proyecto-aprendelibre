Nuestra recomendación de cursos

PYTHON

Este curso empieza desde 0 para aprender a programar con Python, tratando los fundamentos de programación como son las variables y constantes, las condiciones, los bucles y los módulos y funciones, para acabar introduciendo el tratamiento de cadenas de texto y los ficheros. De esta forma, al acabar el curso el alumno será capaz de realizar programas útiles a la hora de procesar datos de diversas fuentes para obtener conclusiones.

FULL STACK JAVA

Descubrí el poder de Java. Aprendé el desarrollo Orientado a Objetos con integración de base de datos. Podrás crear sitios web dinámicos con HTML, CSS y JavaScript y luego integrar FrontEnd y BackEnd mediante la tecnología Java, con ello lograrás convertirte en un programador Full Stack Java.

FULL JAVASCRIPT / NODE.JS

Amplía tus conocimientos en desarrollo web con JavaScript y Node.js. Explora el "detrás de escena" de los sitios web y domina éstas tecnologías en 20 semanas. Impulsá tu carrera en desarrollo web y adquirí las habilidades necesarias para crear aplicaciones web dinámicas y modernas.

PHP

Sumergite en el mundo del desarrollo con PHP, uno de los lenguajes para Back-end presente en el 80% de los sitios web. En este curso de PHP aprenderás de forma práctica los conceptos básicos, las mejores técnicas, así como las librerías más populares y herramientas necesarias para programar de forma eficiente con este lenguaje de programación.

Testing/QA

Aprende las mejores prácticas en evaluación de proyectos informáticos y control de calidad en este curso de 20 semanas. Mejora tus habilidades en el testing automatizado y QA para asegurar la calidad del software. Impulsá tu carrera en este campo fundamental para el desarrollo de proyectos tecnológicos.

DISEÑO UX/UI

Descubrí el apasionante mundo del diseño UX/UI en 20 semanas. Aprendé a crear interfaces atractivas y coherentes que mejoren la experiencia del usuario. Adquirí conocimientos en herramientas de diseño como Figma, Adobe Illustrator y Adobe Photoshop. Podrás desarrollar productos digitales impactantes que se destaquen en el mercado.

BIG DATA/DATA ANALYTICS

Inicia tú recorrido en el análisis de datos y Big Data en 20 semanas. Aprendé a manejar grandes volúmenes de información, crear informes para tomar decisiones en forma práctica y con aplicación laboral. Desarrollá habilidades en data analytics y convertite en un experto en este campo en continuo crecimiento.

Microsoft Power BI Data Analyst

Este curso cubre los diversos métodos y mejores prácticas que están en línea con los requisitos comerciales y técnicos para modelar, visualizar y analizar datos con Power BI. El curso mostrará cómo acceder y procesar datos de una variedad de fuentes de datos, incluidas las fuentes relacionales y no relacionales. Finalmente, el curso también analizará cómo administrar e implementar informes y paneles para compartir y distribuir contenido.

Ciberseguridad

Especializate en una de las habilidades digitales más demandadas en el mercado laboral actual y convertite en el profesional que todas las empresas están buscando. La ciberseguridad es una de las aplicaciones técnicas más demandadas por las empresas en la nueva economía digital. Las compañías están hoy preocupadas por los ciberataques que amenazan sus sistemas informáticos y la falta de profesionales especializados.
